# Трекер прочитанных книг
**Автор:** Румба Альбина

**Описание:**
Простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора и жанра. Реализована фильтрация по жанру и количеству страниц. Данные сохраняются в файл JSON.

**Технологии:**
- Python 3.x.
- Tkinter (GUI).
- JSON (хранение данных).
- Git / GitHub.

## Как запустить приложение?
1. Установите Python 3.x.
2. Клонируйте репозиторий:
   ```
   git clone https://github.com/ВАШ_ЛОГИН/book-tracker.git
   ```
3. Перейдите в папку проекта.
4. Запустите:
   ```
   python main.py
   ```
5. Используйте интерфейс для добавления и просмотра книг.
